# shop_here

A simple ecommerce application for fashionistas to buy different kinds of apparel. This project follows the **flutter_ecommerce_app** repository by **@Santos-Enoque**.

## Features (Work In Progress)
1. Product Overview
2. Product Details
3. Shopping Cart
